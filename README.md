Controls

```
WSAD:      walk
CTRL:      duck
SPACE:     jump
L MOUSE:   fire 
R MOUSE:   toggle auto shotty
Q:         next weapon
V:         next splitscreen mode
F:         toggle fullscreen
1/2/3/4:   switch player
```
